# AllAboutTrento

## Installare npm

```sh
npm install
```
### Per avviare l'applicazione e il server in contemporanea dovremo utilizzare due terminali 

### Avviare l'applicazione

```sh
npm run dev
```

### Per avviare il server dobbiamo spostarci nella cartella /server

```sh
cd .\server\
```

### Per avviare il server

```sh
node .\server.js
```
